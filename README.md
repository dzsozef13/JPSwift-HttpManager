# JPSwiftHttpManager

This package can equip any project with a HttpManager to hangle different Http requests.

Includes:
- HttpManager
    - Compose & Send requests
- Helpers:
    - JSONParser
    - PDFParser
    - URLComposer
- Models:
    - Data & Http Errors
    - HttpContentType
    - HttpRequestType
